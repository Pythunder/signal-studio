Clipboard

The clipboard is the destination for cut and copy, and the source for paste. Signal Studio has a private clipboard, which differs from the Windows clipboard. Signal Studio's clipboard is only accessible to the current instance of Signal Studio, and can't be used to share audio with other applications.

It's also non-persistent, meaning its contents disappear when Signal Studio exits. It's stored on disk by default; see disk threshold.

Note: This is not an option in the Edit Menu. This is only provided for reference.  In general, clipboard's contents are stored in the memory [RAM] and not on the disk.
